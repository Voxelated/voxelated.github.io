Detailed Description

template<typename KeyType, size_t Index, typename TFirst, typename... TRest>
struct Pix::Meta::Detail::FindImpl< KeyType, Index, BasicTuple< TFirst, TRest...> >

Defines the implementation for finding and element in a container.

This case will iterate until either a match is found or there is no match in the container.

Template Parameters
KeyTypeThe type of the key to find in the container.
KeyThe value of the key to try and match.
IndexThe index of the key in the container.
ContainerThe type of the container to search through.
