Detailed Description

template<typename... Types>
struct Pix::Meta::BasicTuple< Types >

Defines a basic tuple class, which essentially is just a cleaner interface for TupleStorage.

The Types are laid out in memory in the same order in which they appear in the parameter pack. For example:

BasicTuple<int, float, double> bt(4, 3.14f, 2.7);
// Laid out as follows:
int <4> // 4 bytes
float <3.14> // 4 bytes
double <2.7> // 8 bytes
