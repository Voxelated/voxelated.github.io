Detailed Description

template<typename Key, typename ValType>
struct Pix::Meta::KeyValue< Key, ValType >

Defines a wrapper to add a key to a type so that it can be used in a Map.

This wrapper is used so that the keys are not actually stored in the Map, but as types, which makes the Map smaller and the lookup constant.

We make the type of the key uint8_t to limit the number of keys for a container. because these containers are compile time, and many of the searches are recursive, the use of uint8_t allows the recursion depth to be limited.

Template Parameters
KeyThe key for the element – this should be unique in that other KeyValue elements used in the same Map must not have the same key.
ValTypeThe type for the value.
