Detailed Description

template<typename... Types>
struct Pix::Meta::Map< Types >

Defines a struct to hold a list of KeyValue types where the value types and instances can be accessed via their keys.

For example:

// Define the keys (not required, but preferred):
enum Keys : uint8_t {
IntKey = 12,
DoubleKey = 24,
StringKey = 7
};
// Define the map:
using SomeMap = Map<
KeyValue<IntKey , int> ,
KeyValue<DoubleKey, double>,
KeyValue<StringKey, string>>;
// Create an instance of the Map:
SomeMap myMap = SomeMap(22, 3.0, "A dummy value");
// We can access the elements like so:
constepxr auto intVal = mpMap.find<IntKey>();
constexpr auto stringVal = myMap.find<StringKey>();

TypesThe types of the elements. These must be KeyValue types.
