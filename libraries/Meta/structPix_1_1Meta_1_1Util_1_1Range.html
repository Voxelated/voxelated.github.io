Detailed Description

template<int64_t Start, int64_t End, int64_t Step = 1, int64_t Current = Start>
struct Pix::Meta::Util::Range< Start, End, Step, Current >

Defines a struct to specify a compile time Range.

It is designed to be used for compile time iteration.

Template Parameters
StartThe start value of the range.
EndThe end value of the range.
StepThe step size over the Range elements.
CurrentThe current value of the range.

static constexpr bool Pix::Meta::Util::Range< Start, End, Step, Current >::isDecreasing ( )
inlinestaticprivate

Checks if the Range is a valid decreasing Range.

The Range is a valid decreasing Range if all of the following conditions are true:

  • The start value is greater than or equal to the end value.
  • The step size is less than than zero.
  • The negative of the step size is a multiple of the range span.
  • The current value doesn't equal the end value.

static constexpr bool Pix::Meta::Util::Range< Start, End, Step, Current >::isValid ( )
inlinestatic

Checks if the Range is valid.

The Range is valid if either of the following sets of conditions are all true:
Incresing range:

  • The end value is greater than or equal to the start value.
  • The step size is greater than zero.
  • The step size is a multiple of the range span.
  • The current value doesn't equal the end value.

Decreasing range:

  • The start value is greater than or equal to the end value.
  • The step size is less than than zero.
  • The negative of the step size is a multiple of the range span.
  • The current value doesn't equal the end value.
