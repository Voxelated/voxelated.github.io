Detailed Description

Defines a struct class to access properties of a Range.

Essentailly this just creates a constexpr type which holds the value of a specific index in a Range, and the size of the Range. For example, if we had the following Range:

using ExampleRange = Range<0, 2>;

Then we would want to create RangeAcess elements as follows:

RangeAccess(0, 2);
RangeAccess(1, 2);
RangeAccess(2, 2);

For a concrete example of the usage, see the implementation of the unrolling functionality.
