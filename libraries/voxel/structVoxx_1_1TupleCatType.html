Detailed Description

template<typename TupleBase, typename TupleNext>
struct Voxx::TupleCatType< TupleBase, TupleNext >

Defines a struct to concatenate two (maybe more) Tuples types into a single Tuple type.

This does not operate on Tuple objects, but rather on Tuple types. This metafuncion is designed to be used to modified the type of the Tuple at compile time. Essentially this allows a mutable compile-time type container.

Template Parameters
TupleBaseThe first tuple to concatenate to.
TupleNextThe second Tuple to concatenate to.
