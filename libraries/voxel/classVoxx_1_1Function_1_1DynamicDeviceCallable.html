Detailed Description

template<typename CallableT, typename... Args>
class Voxx::Function::DynamicDeviceCallable< CallableT, Args >

Implementation of a dynamic device callable wrapper which allocates its arguments on the device.

Template Parameters
CallableTThe type of the callable to wrap.
ArgsThe type of the arguments to wrap.
