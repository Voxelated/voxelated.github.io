§ sharedCores()

static uint32_t Voxx::System::CpuInfo::sharedCores ( uint32_t  level = 0)
static

Returns the number of cores (not threads) shared by the level th level of cache which is shared by cores.

Where level is 0 the number of cores sharing the first level of cache which has core sharing is returned.

I.e if L1 and L2 are only shared by threads on a core, then L3 is the first level of core sharing, and using level = 0 will return the number of cores which share L3.

Parameters
[in]levelThe level for which to get the number of shared cores for.
