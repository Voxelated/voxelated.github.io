Detailed Description

The BitMask class defines a dynamically resizable, bitmask class which can scale to arbitrary size.

Internally, the bitmask is stored contiguously, i.e the first byte is a an 8 wide mask, where the first bit in the byte is flag 0, and the last bit is flag 7. If the required number of bits is a multiple of 8, then the required number of bytes for the class is exactly the number of bits in the mask divided by 8, otherwise there will be totalBits % 8 bits of padding.
