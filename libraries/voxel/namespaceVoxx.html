§ firstSetBitIndex()

constexpr uint64_t Voxx::firstSetBitIndex ( uint64_t  val)
inlinenoexcept

Gets the index of the first set bit, by calculating the log of the value with only the least non zero bit set.

The implementation is taken from:

https://graphics.stanford.edu/~seander/bithacks.html#IntegerLog

Parameters
[in]valThe value to get the index of the least set bit of. Returns the value of the index of the first set bit.
