BitMask Voxx::Thread::getAffinity ( )
noexcept

Gets the affinity for the currently executing thread.

If setAffinity(threadIndex) has been called, this will return a mask with only threadIndex set. Returns a bitmask representing which logical threads the currently executing thread may run on.

◆ setAffinity()

void Voxx::Thread::setAffinity ( std::size_t  coreNumber)
noexcept

Sets the affinity of the currently executing thread so that it executes exclusively on the cpu given by coreNumber.

The core numbers should be user defined. An example useage may be to bind each thread to it's own core:

void run(int coreNumber) {
Voxx::Thread::setAffinity(coreNumber);
// Do work ..
}
int main(int argc, char** argv) {
static constexpr std::size_t threadCount = 8;
std::vector<std::thread> threads(threadCount);
for (int i = 0; i < threads.size(); ++i)
threads[i] = std::thread(run, i);
for (auto& thread : threads)
thread.join();
}

[in]coreNumberThe number of the core to set the thread to run on.
