Detailed Description

template<typename CallableType, typename... Args>
class Voxx::Function::DynamicCallable< CallableType, Args >

This class defines functionality for wrapping a callable object which dynamically allocates the arguments arguments which may change (i.e they may be modified on the device, and are non-const lvalue reference types and hence we need to get the modified versions back from the device onto the host).

Template Parameters
CallableTypeThe callable object to invoke.
ArgsThe types of the arguments.
