Detailed Description

template<typename... Types>
struct Voxx::Tuple< Types >

Defines a struct to hold a fixed-size number of heterogenous types.

Specialization for a non-empty Tuple.

This implementation of Tuple is similar to std::tuple, but allows the tuple to be used on both the host and the device. We could use thrust::tuple, however, it only supports 9 types (rather than the compiler's recursion depth limit in out case), since it doesn't use variadic tempates, which is also limiting.

The interface through which a tuple should be created is the make_tuple() function, i,e:

auto tuple = make_tuple(4, 3.5, "some value");

unless the tuple needs to store reference types. See the documentation for make_tuple for an example.
