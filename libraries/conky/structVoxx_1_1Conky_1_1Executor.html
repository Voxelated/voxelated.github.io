Detailed Description

template<typename CallableType, typename... Args>
struct Voxx::Conky::Executor< CallableType, Args >

The Executor class is used to store a generic executor, and is simply a wrapper around a Callable which derives from TaskExecutor which allows it to be stored in and run by a Task.

CallableTypeThe type of the callable to store.
