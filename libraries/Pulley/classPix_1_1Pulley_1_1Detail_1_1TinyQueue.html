Detailed Description

template<typename Type, template< class > typename AtomicType>
class Pix::Pulley::Detail::TinyQueue< Type, AtomicType >

The TinyQueue class defines a blocking queue which can store only a single element.

Users of the queue are required to assign turns (using unique identifiers) when pushing and popping to/from the queue.

The queue can be used as follows:

Template Parameters
TypeThe type of the queue elements.
AtomicTypeThe Type of the atomic used for concurrent scheduling.
