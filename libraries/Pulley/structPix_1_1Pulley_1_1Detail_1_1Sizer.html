Detailed Description

template<typename... Types>
struct Pix::Pulley::Detail::Sizer< Types >

Defines an object to determine the size of a union of types.

The union should always have the size of the type of maximum size. We use the MaxSizeOf function from Meta to scan the type parameter pack and return the size of the maximum element.

Template Parameters
TypesThe types from which to find the size of the biggest type.
