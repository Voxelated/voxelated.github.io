Detailed Description

template<typename SchedulerImpl, template< class > typename AtomicWrapper = std::atomic>
class Pix::Pulley::Scheduler< SchedulerImpl, AtomicWrapper >

The Scheduler class defines the interface for all scheduling objects.

It uses CRTP to specialize on the scheduling type, allowing different schedulers to be interchanged at compile time.

Template Parameters
SchedulerImplThe implementation of the specific scheduling type.
AtomicWrapperThe wrapper around a variable to make it atomic.

Checks if a thread will execute.

What this means is implementaion specific, for example, a sequential scheduling implementation may return true if a specific ticket will execute, while an out of order scheduler may return true if any thread may execute.
