Detailed Description

The ConditionVariable class is a synchronization primitive which can be used to block one or more threads at a time, until some other thread modifies a shared variable (the condition) and notifies the ConditionVariable.

noexcept

Causes the current thread to block until the ConditionVariable is either notified, or a spurious wakeup occurs.

Allows a predicate to be provided to determine if waiting must continue if a spurious wakeup occurs. This is equivalent to:

while (!predicate()) {
conditionVariable.wait(mutex);
}
Parameters
[in]mutexThe mutex which locks the thread.
[in]predicateParameter to determine if attempting to wait must continue on spurious wakeup. This must have the following signature: bool predicate().
Template Parameters
PredTypeThe type of the predicate.
