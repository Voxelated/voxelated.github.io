Detailed Description

The BlockFormatter is a specialization of the Formatter<> CRTP class which defines the indentation and the separator fort the formatting.

Using the specialization, the following use of the API from within a specialization of a BasicError<>:

template <typename FormatterImpl>
void logImpl(OStream& os, Formatter<FormatterImpl>& formatter) {
//==--- Required by BasicError<> conventions: ---------------------==//
// The type and error code are printed:
formatter.appendNewInfo(os, std::string("Logic Error: ") +
errorTypeCodeToString(this->ErrorCode));
// The location is then printed:
formatter.appendNewInfo(os, this->Location);
//==--- Optional --------------------------------------------------==//
// This is extra info:
formatter.appendNewInfo(os,
std::string("Extra error info which is really long ... and will") +
std::string("span many lines ... will be auto formatted"));
formatter.appendNewInfo(os, "Some final info");
}

Would produce the following output:

{}
Output unrelated to the error ...
==-----------------------------------------------------------------------
| LogicError: OutOfRangeAccess
==-----------------------------------------------------------------------
| at -> someExampleFunction()
==---------------------------------------------------------------------
| Extra error info which is really long .. and will span many lines
| ... will be auto formatted
==-------------------------------------------------------------------
| Some final info
==-----------------------------------------------------------------

The formatting makes information easy to find, and to understand.
