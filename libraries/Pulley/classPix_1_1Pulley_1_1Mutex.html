Detailed Description

The Mutex class is a synchronization primitive which is used to protect shared data from being accessed by multiple threads simultaneously.

Member Enumeration Documentation

enum Pix::Pulley::Mutex::States : LargeType
private

Defines the states which the mutex can be in, when looking at the whole word of the state.

The upper byte is used for stating contention (0 | 1), and the lower byte is used for stating lockedness (0 | 1).
