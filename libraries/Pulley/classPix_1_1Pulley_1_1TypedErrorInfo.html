Detailed Description

template<typename ErrorType, typename ParentErrorType = TypedErrorInfoBase>
class Pix::Pulley::TypedErrorInfo< ErrorType, ParentErrorType >

The TypedErrorInfo class defines a base class is the interface which newly defined error typed must conform to.

For example:

class CustomError : public TypedErrorInfo<CustomInfo> {
// Custom error functionality ...
};
Template Parameters
ErrorTypeThe type of the error which will be defined.
ParentErrorTypeThe type of error from which this error is derived.
